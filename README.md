# Neural ODE 기반 저지구 궤도(LEO) Kessler Syndrome 예측

본 프로젝트는 Neural ODE를 활용하여  
- 위성–파편 상호작용(S–D 시스템) 동역학 재현  
- Cascade Phase(연쇄 충돌 지속 구간) 탐지  
- 계산 효율성 및 다중 초기 조건 평가 성능 비교  
를 수행하였다.

## 연구 참여자
본 연구는 서울과학고 2학년 학생들이 공동으로 수행하였다.
- **박주혁**
- **이서준**
- **임우석**
- **지승연**

**지도교사**: 이선민 (서울과학고 지구과학 전공)  
**지도교수**: 허태민 (홍익대학교 건설환경공학과)
